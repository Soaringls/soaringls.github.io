基本流程：

1. 分割地面点和非地面点
2. 基于非地面点进行点云分割
3. 实时加载hdmap将位于道路上的分割结果进去滤出(亦可通过tracking将动态障碍物滤除)

<!-- more -->
#### 整体概览
1. 左下角为原始点云数据
2. 右下角为检测的地面点云
3. 左上角为分割的结果(已根据hdmap滤出位于道路上的动态障碍物)
4. 右上角为滤出动态障碍物后点云
